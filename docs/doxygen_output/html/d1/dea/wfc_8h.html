◆ get_fft_box()

void get_fft_box ( const ELPH_float EcutRy,
const ELPH_float * blat,
ND_int * fft_box,
MPI_Comm commK )

Calculate FFT box (Nx, Ny, Nz). The dimensions are determined by:

  1. The energy cutoff (EcutRy) which defines the maximum kinetic energy.
  2. Each cpu will atleast have one Z stick i.e Nx * Ny > comm_size. So this is cpu dependent

The initial dimensions are calculated as: N_i = 2*round(sqrt(Ecut)/|b_i| + 1) + 1 If the initial grid would be too small for parallelization (Nx*Ny < num_processes), the grid is scaled up by a factor to ensure each process gets at least one Gz stick.

Parameters
[in]EcutRyEnergy cutoff in Rydberg units
[in]blatArray of reciprocal lattice vectors (9 elements, 3x3 matrix)
[out]fft_boxArray to store the computed FFT box dimensions (Nx, Ny, Nz)
[in]commKMPI communicator for k-point parallelization
Attention
The function uses MPI Bcast so it must be called by all processes in commK
