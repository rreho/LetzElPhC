LetzElPhC Documentation

Introduction

LetzElPhC (Lëtzebuerg Electron-Phonon Code) is a high-performance computational physics software designed for calculating electron-phonon coupling matrix elements from first-principles calculations.

The code processes output from Quantum ESPRESSO (QE) DFT calculations and computes the electron-phonon interaction matrices needed for studying various electron-phonon phenomena in condensed matter systems.

Key Features

  • Direct calculation of electron-phonon matrix elements
  • Parallelized with MPI and OpenMP for large-scale calculations
  • Efficient Wannier interpolation of electron-phonon matrices
  • Currently supports Quantum ESPRESSO
  • Exploits crystal symmetries for computational efficiency
  • Supports spin-orbit coupling and magnetic systems

Important Notes

This code computes only electron-phonon matrix elements. For calculating derived physical properties (mobilities, superconducting properties, etc.), use complementary tools such as the EPW code.

The code is currently under active development. Please use at your own risk and report any issues encountered.
