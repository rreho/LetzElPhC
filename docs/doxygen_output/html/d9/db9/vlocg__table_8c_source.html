64 value of |G+q| |G+q| <= |G| + |q|. If all three values of q (in crystal
65 coordinates) are restricted to [qmin,qmax] then |q| <= 3*b_max*|qmax|
66 G_max = |N1/2*b1 + N2/2*b2 + N3/2*b3| < N_max/2 * 3 * b_max
