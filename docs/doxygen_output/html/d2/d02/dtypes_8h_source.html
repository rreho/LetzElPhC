287 /*
288 for Example if there are total 8 cpus in MPI_COMM_WORLD
289 divide as 2 qpools and 2 kpools
290
291 Global {0,1,2,3,4,5,6,7}
292 _________________|__________________
293 | |
294 {0,1,2,3} (commQ) {4,5,6,7} (commQ)
295 _________|__________ _________|__________
296 | | | |
297 {0,1}(commK) {2,3}(commK) {4,5} (commK) {6,7} (commK)
298
299 npw_cpus = size of commK
300 commR contains {0,2,4,6} and {1,3,5,7} i.e rank%npw_cpus is used to make
